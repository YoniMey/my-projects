My Thesis (2025)

Goal: Build an LLM-based agent system that answers and reasons about complex temporal questions using RDF graphs.

Core Idea: Convert a document into a time-focused RDF knowledge graph and query it with SPARQL. DSPy is used for LLM orchestration and prompt optimization.

  • OWL Schema Generation: A DSPy-optimized LLM generates OWL schemas defining classes and relations.
  • Entity Extraction & Linking: GLiNER extracts entities; ReLiK links them and identifies relationships.
  • RDF Graph Construction: LLMs iteratively propose graphs, validated with rdflib, improved via SPARQL-based QA.
  • SPARQL Query Generation: Spinach agents learn the graph structure to produce queries from natural language.

AudioCorrect (2025)

Concept: A speech editing app that lets users correct words in a recording by editing the transcript.

  1. Transcribe with WhisperX and extract word timings.
  2. Mark sections to edit (insert/delete/replace).
  3. Generate audio with Tortoise TTS or ElevenLabs API.
  4. Assemble audio with fades, silence padding, and volume matching.

ReDress (2023)

2nd Place - Samsung NEXT Hackathon for Mobile Generative AI

Idea: A marketplace app for second-hand clothing powered by CLIP embeddings.

  • Buyers: Use natural language to search. CLIP maps it to image embeddings for retrieval—no filters needed.
  • Sellers: Upload an image and price. CLIP handles categorization and description automatically.
